# Apache Sling Servlet Examples

Collection of real-world Apache Sling servlet implementations (as provided in this chat).

## Files Included

1. **MyGetServlet.java** — SlingSafeMethodsServlet - Basic GET Servlet
2. **PathBasedServlet.java** — Path-Based Servlet with OSGi DS Annotations
3. **JsonResourceServlet.java** — JSON Response Servlet
4. **FormHandlerServlet.java** — POST Servlet with Form Handling
5. **RestApiServlet.java** — All Methods Servlet
6. **ServiceInjectionServlet.java** — Servlet with Service Injection
7. **ResourceInclusionServlet.java** — Resource Resolution Servlet
8. **MultiSelectorServlet.java** — Selectors and Extensions Servlet

## Notes

- Package declarations are preserved as-is (`com.example.core.servlets`).
- Some examples reference additional services/dependencies (e.g., `DataService`, Commons IO).
